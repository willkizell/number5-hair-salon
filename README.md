# Number 5 Hair Salon — Website

## Setup

1. Drop `logo.png` into this root folder (next to `index.html`)
2. Drop photos into the `/photos` folder:
   - `photos/hero.jpg` — hero section (portrait orientation, salon interior or stylist at work)
   - `photos/interior.jpg` — about section (portrait orientation, salon space)
   - `photos/salon.jpg` — optional extra

All photos gracefully fall back to gradient placeholders if the files aren't present yet.

The logo renders in dark mode via CSS filter — no separate dark logo file needed.
